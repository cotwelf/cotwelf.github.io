关于提示 React Hook useEffect has a missing dependency 的问题

2022-04-24 前端 React

假设我们有个需求，页面刚进入的时候放个定时器，1s 后 count + 1。我是这么干的

import React, { useEffect, useState } from 'react'
import * as ReactDOM from 'react-dom'

const DiyComponent = () => {
  const [count, setCount] = useState(0)
  const printCount = () => {
    setCount(count + 1)
  }
  useEffect(() => {
    const counter = setTimeout(() => {
      printCount()
    }, 1000)
    return () => {
      clearTimeout(counter)
    }
  }, [])
  return (
    <div>{count}</div>
  )
}

ReactDOM.render(<DiyComponent />, document.getElementsByClassName('container')[0])

这样页面看起来是可以正常显示，但有个 lint 问题

本来想 disabled 掉，但还是跟着 lint 的提示这样改了下

打开页面一看，timeout 居然开始计数了，怎会如此？

const [count, setCount] = React.useState(0);
const printCountRef = React.useRef(null);
const printCount = React.useCallback(() => {
  setCount(count + 1);
}, [count]);
printCountRef.current = printCount;
React.useEffect(() => {
  const counter = setTimeout(() => {
    printCountRef.current();
  }, 1000);
  return () => {
    clearTimeout(counter);
  };
}, []);
return <div>{count}</div>;
