Date.now()、+new Date() 以及 Date 创建的实例的 valueOf()

let a = Date.now()
let b = +new Date()
let c = new Date()
let d = c.valueOf()

其中，abd 是相等的。因为 Date 实例的 valueOf 返回的是 1970 年 1 月 1 日午夜（零时） 开始经过的毫秒数，所以 Date 实例之间是可以比较的（不用转毫秒数）

支持 Date.now()方法的浏览器包括 IE9+、Firefox 3+、Safari 3+、Opera 10.5 和 Chrome。在不支持它的浏览器中，使用 + 操作符把 Date 对象转换成字符串，也可以达到同样的目的。

常用的 Date 方法

方法 说明
getFullYear() 取得4位数的年份
getMonth() 返回日期中的月份，其中0表示一月，11表示十二月
getDate() 返回日期月份中的天数（1到31）
getDay() 返回日期中星期的星期几（其中0表示星期日，6表示星期六）
getHours() 返回日期中的小时数（0到23）
getMinutes() 返回日期中的分钟数（0到59）
getSeconds() 返回日期中的秒数（0到59）

原生 Date 方法不多，比如无法格式化日期、计算两个时间的时间差等，如果想用更多的方法可以使用第三方库，比如 moment 和 dayjs 都很不错，dayjs 相比 moment 更轻量，但功能没有 moment 强大，不过基本常用的方法还是有的，可以根据项目需要进行选择。

正则表达式

匹配模式

g：全局；i：不区分大小写；m：多行

例如：/at/gi

需要转义的元字符：( [ { \ ^ $ | ) ? * + .]}

// 匹配所有以"at"结尾的 3 个字符的组合，不区分大小写
var pattern1 = /.at/gi;
'cat.at'.match(pattern1) // ['cat', '.at']
'cat.at'.search(pattern1) // 0 （匹配到了 cat

// 匹配所有".at"，不区分大小写
var pattern2 =  new RegExp("\\.at", "gi"); // 不能把正则表达式字面量传递给 RegExp 构造函数
'cat.at'.match(pattern2) // ['.at']
'cat.at'.search(pattern2) // 3

// 加了全局匹配 g 的话会循环，最后一次是 null，之后又会重头开始
var pattern2 =  new RegExp("at", "gi");

// 若不设置全局则始终返回第一个匹配项
pattern2.exec('cat.at') // ['at', index: 1, input: 'cat.at', groups: undefined]
pattern2.exec('cat.at') // ['at', index: 4, input: 'cat.at', groups: undefined]
pattern2.exec('cat.at') // null
pattern2.exec('cat.at') // ['at', index: 1, input: 'cat.at', groups: undefined]

pattern2.test('cat.at') // true
pattern2.test('cat.at') // true
pattern2.test('cat.at') // false

// match 方法例外，不论是否全局都会返回所有匹配项
'cat.at'.match(pattern2) // ['at', 'at']
'cat.at'.match(pattern2) // ['at', 'at']
'cat.at'.match(pattern2) // ['at', 'at']
